# Employee Test

## Prerequisites
Before you can run the project, make sure you have the following installed:
- [Java 17](https://www.oracle.com/java/technologies/javase-jdk17-downloads.html)
- [MySQL](https://www.mysql.com/downloads/)

### MySQL Setup
1. Install MySQL and start the MySQL server.
2. Create a database for the project using the following command:
    ```sql
    CREATE DATABASE your_database_name;
    ```
3. Update the `application.properties` (or `application.yml`) file with your MySQL credentials:
    ```properties
    spring.datasource.url=jdbc:mysql://localhost:3306/your_database_name
    spring.datasource.username=your_username
    spring.datasource.password=your_password
    ```

## Running the Project
Once you've set up the prerequisites:
1. Clone the repository:
    ```bash
    git clone https://github.com/KyawThet23/Java-code-test
    ```
2. Navigate to the project directory:
    ```bash
    cd test
    ```
3. Build the project using Maven or Gradle and run

## Swagger API Documentation
